# Code of Conduct

We are committed to a welcoming, harassment-free community for contributors of all backgrounds.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on code and outcomes, not people.
- Assume good intent and ask clarifying questions.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Personal attacks or abusive language.
- Deliberate disruption of community collaboration.

## Reporting

Report concerns privately to maintainers via repository security/contact channels.
